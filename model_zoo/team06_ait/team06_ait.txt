You can also override values by environment variables:
- SEESR_INFER_ROOT
- SEESR_PRETRAINED_MODEL_PATH
- SEESR_MODEL_PATH
- SEESR_RAM_FT_PATH
- SEESR_USE_DEGRADATION_TOKEN
- SEESR_DEGRADATION_ADAPTER_PATH
- SEESR_USE_DYNAMIC_DEGRADATION_TOKEN

Folder naming tip:
- You can rename the backend folder freely (e.g., `models/team06_AIT/my_backend`),
  then set `"inference_root": "models/team06_AIT/my_backend"` in `team06_ait.json`.

Current default:
- `"inference_root": "models/team06_AIT/ait_backend"`
